# Seller App

> The best Seller app of world

## Getting Started

Install dependecies:

```shell
yarn
```

Start the dev API server (powered by JSON server):

```shell
yarn start:api
```

start the dev servers :

```shell
yarn start
```

Make your changes 👩‍💻.

Commit your feature:

```shell
yarn commit
```
